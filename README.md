# Misitas Webshop

Een moderne webshop voor Misitas.

## Project Structuur

```
misitas/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── main.js
└── assets/
    └── images/
```

## Getting Started

1. Clone de repository:
   ```bash
   git clone https://github.com/jouw-gebruikersnaam/misitas.git
   ```

2. Open `index.html` in je browser om te starten.

## Tech Stack

- HTML5
- CSS3
- JavaScript
